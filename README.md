Join – Project Management Tool
A collaborative project management web application built with a team of developers using modern web technologies.

About the Project
Join is a Kanban-style project management tool developed as part of a coding bootcamp in a team of four developers.
The goal of the project was to design and build a fully functional web application that allows users to organize tasks, track progress, and manage responsibilities efficiently.

Team Collaboration

This project was built in a team of 4 developers, focusing on:
- Agile workflows
- Task distribution
- Git-based collaboration (branches, pull requests, code reviews)
- Regular coordination and feature integration

Features
- Task Management
- Create, edit, and delete tasks
- Assign tasks to users
- Set priorities and due dates
Kanban Board
- Drag & drop tasks between columns
- Status tracking (To Do, In Progress, Done)
User Assignment
- Assign tasks to team members
- Visual representation of responsibilities

Technologies Used

HTML5
CSS3
JavaScript (ES6)
Firebase (Database / Authentication – ggf. anpassen)
Git & GitHub (version control & collaboration)
